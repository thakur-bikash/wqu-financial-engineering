# WQU MScFE Course Directories

This directory contains folders for the WorldQuant University Master of Science in Financial Engineering (MScFE) courses. The naming convention uses the course number followed by an acronym of the course title.

## Directory Map

* **101dip**: MScFE 101 Workshop: Delivering Impactful Presentations
* **560fm**: MScFE 560 Financial Markets
* **600fd**: MScFE 600 Financial Data
* **610fe**: MScFE 610 Financial Econometrics
* **620dp**: MScFE 620 Derivative Pricing
* **622sm**: MScFE 622 Stochastic Modeling
* **632mlf**: MScFE 632 Machine Learning in Finance
* **642dlf**: MScFE 642 Deep Learning for Finance
* **652pm**: MScFE 652 Portfolio Management
* **660rm**: MScFE 660 Risk Management
